# Siddartha Bandi — Portfolio

Personal portfolio website for **Siddartha Bandi**, Data Scientist & AI Engineer (MS Data Science @ University of North Dakota).

**Live site:** [bandisiddartha.github.io/siddartha-portfolio](https://bandisiddartha.github.io/siddartha-portfolio/)

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | Next.js 16 (static export) |
| Styling | Tailwind CSS v4 |
| Animations | Framer Motion |
| Language | TypeScript |
| Deployment | GitHub Pages via GitHub Actions |

---

## Sections

- **Hero** — Two-column split with profile photo, typewriter role animation, and CTA buttons
- **About** — Bio, quick facts, and domain expertise cards
- **Technical Skills** — Animated proficiency bars and full tech badge cloud
- **Featured Projects** — Satellite imagery pipeline, medical image analysis, FinTech risk models
- **Contact** — LinkedIn, GitHub, and email links

---

## Running Locally

```bash
npm install
npm run dev
```

Open [https://bandisiddartha.github.io/siddartha-portfolio/](https://bandisiddartha.github.io/siddartha-portfolio/).

## Building

```bash
npm run build   # generates ./out static export
```

---

## Deployment

Pushing to `main` automatically triggers the GitHub Actions workflow (`.github/workflows/deploy.yml`) which builds the static site and deploys it to GitHub Pages.
